

# Portofolio Proyek EE Sistem Tenaga Listrik

Website ini adalah portofolio proyek untuk PT. Prana Bhaskara Utama yang menampilkan daftar project dalam bentuk embed web (iframe) dari project lain.

## Fitur Utama
- Menampilkan daftar project dalam bentuk embed web (iframe) dari project lain.
- Setiap project ditampilkan sebagai kartu dengan judul, link ke project, dan tampilan embed langsung.
- Loading animasi pada embed project.
- Tampilan responsif dan modern.
